## Project Description
This project is a ticketing system for eMagiz to manage:
- Alerts
- Incidents
- Requests for Change (RFC)

## Features
### Implemented
- Secure login & registration (Argon2id password hashing)
- Role-based access (Customer / Consultant / Support)
- Ticket creation, listing, and viewing
- Ticket categorization (Incident / RFC) and priority levels
- Dashboard overview with search & filtering
- SQL persistence
- Audit Logging
- More on Trello: https://trello.com/b/L6mDl96V/emagiz-2
- Deployed: https://emagiz2.paas.hosted-by-previder.com/
